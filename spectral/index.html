    jaxdf.spectral.fft_interp(k: ndarray, spectrum: ndarray, x: ndarray) -> float

    Calculates the value of a field fff at an arbitrary position xxx using

    f(x)=∑if^eik⊤x f(x) = \sum_i \hat f e^{ik^\top x} f(x)=i∑​f^​eik⊤x

    for complex signals. The spectrum and the frequency coordinate grid kkk must conform to jax.numpy.fft.fftn.
