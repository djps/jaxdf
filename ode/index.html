    jaxdf.ode

    jaxdf.ode.euler_integration(f, x0, dt, output_steps, timestep_correction=1)

    Integrates the differential equation

    x˙=f(x,t) \dot x = f(x,t) x˙=f(x,t)

    using a first-order Euler method. The solution xxx is given by

    x(n+1)=x(n)+f(x(n),t(n))dt⋅κ x^{(n+1)} = x^{(n)} + f\left(x^{(n)}, t^{(n)}\right)dt\cdot\kappa x(n+1)=x(n)+f(x(n),t(n))dt⋅κ

    The structure of xxx is inferred by x(0)x^{(0)}x(0), which can be any pytree. The output of fff should be compatible with x(0)x^{(0)}x(0) (must have the same pytree structure)

    output_steps must be the index of the steps to save. κ\kappaκ is timestep_correction.

    The stepsize of the euler integration is given by κ⋅dt\kappa\cdot dtκ⋅dt.

    Example

    from jaxdf.ode import euler_integration

# Simulation settings
g = 9.81
dt = 0.01
output_steps = jnp.array([0,5])/dt # Must be in steps, not seconds

# Newton law of motion
f = lambda (x, v), t: v, 0.5*g

# Initial conditions
x0 = 1.
v0 = 0.

# Calculate endpoint
x_end, v_end = euler_integration(f, (x0,v0), dt, output_steps)

    jaxdf.ode.generalized_semi_implicit_euler(params, f, g, measurement_operator, alpha, x0, y0, dt, output_steps, backprop=False, checkpoint=False)

    This functions works in the same way as the semi_implicit_euler integrator, with the difference that the update function accepts an extra parameter α\alphaα with the same pytree-structure as xxx and yyy.

    Variable update is performed as

    {x(n+1)=α[αx(n)+f(y(n),t(n))dt⋅]y(n+1)=α[αy(n)+g(x(n+1),t(n))dt⋅]r(n)=M(x(n+1),y(n+1)) \begin{dcases} x^{(n+1)} &= \alpha\left[\alpha x^{(n)} + f\left(y^{(n)}, t^{(n)}\right)dt\cdot\right] \\ y^{(n+1)} &= \alpha\left[\alpha y^{(n)} + g\left(x^{(n+1)}, t^{(n)}\right)dt\cdot\right] \\ r^{(n)} &= M(x^{(n+1)}, y^{(n+1)}) \end{dcases} ⎩⎨⎧​x(n+1)y(n+1)r(n)​=α[αx(n)+f(y(n),t(n))dt⋅]=α[αy(n)+g(x(n+1),t(n))dt⋅]=M(x(n+1),y(n+1))​

    M(x,y)M(x,y)M(x,y) is an arbitrary measurement operator that is applied at the end of each timestep, for example it could evaluate the pressure intensity or the field value at some specific locations. If None, defaults to the identity operator I(x,y)=(x,y)I(x,y)=(x,y)I(x,y)=(x,y). The vector of measurements rrr=r is returned.

    Warning

    Calling this method with backprop=True allows to perform backpropagation. However, this requires storing the entire wavefield history and is therefore memory demanding. Alternatively, backprop=False allows to calculate derivatives using forward-propagation, or jacobian-vector products with memory cost independent of the simulation length. Combined with jax.vmap or jax.jaxfwd, this makes easy to calculate gradients for functions that have tall jacobians, such as simulations that depends on a small amount of parameters (e.g. delays, steering angle, etc)

    Example

    # Integrating the equations of motions for a planet around a star
M_sun = 2.0  # kg
p0 = jnp.array([0.0, 3.0])  # m
v0 = jnp.array([1.0, 0.0])  # m/s
G = 1
dt = 0.1
t_end = 200.0
output_steps = (jnp.arange(0, t_end, 10 * dt) / dt).round()

# Equations of motion
f_1 = lambda v, t: v
f_2 = lambda p, t: newton_grav_law(G=1, M=M_sun, r=p)

M = lambda x: x # Identity operator, could have been `None`

# Integrate
trajectory , _ = generalized_semi_implicit_euler(
    f = f_1, 
    g = f_2, 
    measurement_operator = M, 
    alpha=0.0,
    x0 = p0,
    y0 = v0,
    dt = dt,
    output_steps = output_steps,
    backprop=False
)

    jaxdf.ode.semi_implicit_euler(f, g, x0, y0, dt, output_steps, timestep_correction=1)

    Integrates a system of differential equations having the form

    {x˙=f(y,t)y˙=g(x,t) \begin{dcases} \dot x = f(y,t) \\ \dot y = g(x,t) \end{dcases} {x˙=f(y,t)y˙​=g(x,t)​

    The solution is computed using a first-order semi-implicit Euler integrator, which almost conserves the energy for time-independent equations, as follows

    {x(n+1)=x(n)+f(y(n),t(n))dt⋅κy(n+1)=y(n)+g(x(n+1),t(n))dt⋅κ \begin{dcases} x^{(n+1)} &= x^{(n)} &+& f\left(y^{(n)}, t^{(n)}\right)dt\cdot \kappa \\ y^{(n+1)} &= y^{(n)} &+& g\left(x^{(n+1)}, t^{(n)}\right)dt\cdot \kappa \end{dcases} ⎩⎨⎧​x(n+1)y(n+1)​=x(n)=y(n)​++​f(y(n),t(n))dt⋅κg(x(n+1),t(n))dt⋅κ​

    The structure of xxx is inferred by x(0)x^{(0)}x(0), which can be any pytree. The output of fff should be compatible with x(0)x^{(0)}x(0) (must have the same pytree structure). A symmetric argument holds for ggg.

    output_steps must be the index of the steps to save. κ\kappaκ is timestep_correction.

    The stepsize of the euler integration is given by κ⋅dt\kappa\cdot dtκ⋅dt.
